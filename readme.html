공지사항

홍익수강

홍익수강은 홍익대학교의 공식 웹사이트가 아닙니다. 실제 수강신청 페이지와 다를 수 있습니다. 이로 인해 발생하는 모든 일에 제작자는 책임지지 않습니다.

"인터넷 익스플로러" 브라우저와 "사파리" 브라우저에서 사소한 버그가 있습니다.
"크롬" 브라우저 이용을 권장드립니다.

모든 종류의 피드백을 환영합니다. 피드백을 hongiksugang@gmail.com 으로 보내주세요.

수강신청 연습

수강신청 연습 페이지는 2월과 8월에만 이용 가능합니다.

모든 학생은 듣고 싶은 수업을 들을 수 있어야 합니다. 홍익대학교에 충분한 수업 개설을 요구합니다.

학교 지도

2011년 홍익대학교 시각디자인과 RRRproject에서 만든 "홍그와트 지도"를 참고하여 만들었습니다.
